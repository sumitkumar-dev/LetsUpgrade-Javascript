Question 2: Explain with examples the remaining methods of String and Array

(A) Methods of string:

1. String Length -It returns the length of the String

Example:

var text = "Harry had a little lamb";

console.log(text.length)

2. Search String - It returns the first occourence of the specified text.

Example:

var text = "Harry had a little lamb";

console.log (text.indexOf('lamb'))

3. Slice String - extracts a part of a string and returns the extracted part in a new string.

Example:

var text = "Harry had a little lamb";

console.log (text.slice(10,14))

4. Replacing String Content - It replaces a specified value with another value in a string.

Example:

var text = "Harry had a little lamb";

console.log (text.replace("lamb","dog"))

(A) Methods of Arrays:

1. Array to String -It converts an array to a string

Example:

var animals = ["Dog", "Cat", "Cow"]

console.log (animals.toString())

2. Popping -It removes the last element from an array

Example:

var animals = ["Dog", "Cat", "Cow"]

console.log (animals.pop())

3. Pushing -It returns the new array length:

Example:

var animals = ["Dog", "Cat", "Cow"]

console.log (animals.pop())

4. Pushing -It returns the string that was "shifted out"

Example:

var animals = ["Dog", "Cat", "Cow"]

console.log (animals.shift())
