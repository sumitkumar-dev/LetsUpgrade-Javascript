Question 1 : Explore and explain the various methods in console function & Explain them.

Some examples of console functions:

1. console.log - It is a function that writes a message to log on the debugging console but you won't see anything on the browser screen

2. console.error - It is used for reporting errors. It reports JavaScript-related errors & CSS errors.

3. console.warn - It is used to write warnin to the console .

Question 2 : Write the difference between var, let and const with code examples

1. var - They are GLOBAl or FUNCTION scope. Their vale can be updated or redeclared and the program will run smoothly.

Example:

var friend = {

name : "Sumit",

}

console.log(friend);


2. let - They have block scope. They can only be updated and cannot be redeclared

Example:

{

let age = "24";

console.log (age);

}


3. const - They also have a block scope but their value cannot be updated or redeclared

Example:

const Nationality = "Indian";

console.log (Nationality);
